# Jiangsu Library Proposal Rules

## Source Context

This reference is tailored to the `2026` Jiangsu higher-education library committee teaching-reform call.

Core source:

- 江苏省高等学校图书情报工作委员会《2026 年江苏省高校图工委教改研究课题立项建设申报工作的通知》

## What Matters Most

### 1. Topic fit is not optional

The proposal should clearly map to one or more of these 2026 directions:

- knowledge-service innovation for new quality productive forces
- cultural育人 practice
- AI literacy and metacognition cultivation
- librarian digital competence
- red literature / special collections innovation
- AI-driven future learning center
- AI-assisted reading promotion
- “三全育人” role of libraries
- reading promotion for “书香江苏”
- transformation and optimization of learning-support services in smart libraries

### 2. Jiangsu reviewers will reward complete implementation logic

The notice explicitly stresses:

- clear goals
- clear ideas
- complete research and implementation plan
- explicit expected outcomes
- practical significance
- promotion value

### 3. Matching funds are a hard constraint

Suggested matching funds in the call:

- key project: `2 万/项`
- general project: `1 万/项`

No matching funds means no project approval.

### 4. Category choice must be realistic

- `重点课题` is normally for stronger foundations; the host should in principle hold a senior title
- `一般课题` is often the safer category if the proposal is good but the foundation is not yet strong enough

### 5. Applicant suitability matters

Important restrictions include:

- only Jiangsu university library staff may apply
- one host only
- one host may apply for only one project
- applicants with unresolved previous committee projects may not apply
- if last year's library fact data were not reported, approval is blocked

### 6. Reviewer lens

The strongest Jiangsu proposals usually do four things at once:

- they fit the annual theme exactly
- they sound like teaching reform rather than procurement
- they show a believable implementation path
- they make the推广价值 easy to imagine

## Audit Questions

Use these questions during review:

1. Does the title immediately fit the 2026 call?
2. Is the project framed as library teaching-reform/service reform rather than pure system construction?
3. Is the implementation path complete enough to satisfy the notice?
4. Is matching funding explicitly committed?
5. Is the category choice realistic?
6. Are outputs concrete enough for recommendation and later acceptance?
7. Is there enough prior basis to believe the team can deliver?
